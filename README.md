# DATN
Kiểm thử tự động cho website quản lý nhân sự OrangeHRM Live Demo
**Giới thiệu**
Dự án này là hệ thống kiểm thử tự động dành cho trang web OrangeHRM Live Demo, sử dụng các công nghệ:
- Java
- Selenium WebDriver
- TestNG
- ExtentReports
- Log4j2
- WebDriverManager
- Quản lý phụ thuộc bằng Maven
